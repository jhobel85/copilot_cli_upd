# CLI Mastery

Purpose:
- Provide general-purpose guidance, tips, and helpers for working with command-line tools and shell workflows.

HARD-GATE:
- Do not run or suggest any destructive commands without explicit user confirmation.

Usage:
- Offer safe, concise examples for common tasks (file navigation, searching, git basics, PowerShell snippets).
- Prefer explanation first, then an example. When suggesting commands that change state, include a confirmation step.

Notes:
- This is a general, user-level skill intended to remain in user scope.
