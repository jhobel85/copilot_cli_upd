# copilot-project-scanner plugin

Tento plugin obsahuje sadu agentů (.md) pro automatické prohledávání a analýzu projektů.
Struktura:
- agents/: jednotlivé agent .md soubory
- mcp/: šablony pro MCP servery
- examples/: příklady pro Python a JS

Instalace a použití: zkopírujte složku .copilot-plugin do kořene repozitáře; CLI by měl plugin načíst automaticky.
