# Scripts

Standalone utility scripts — run directly, no install step.

| Script | Platform | Description |
|--------|----------|-------------|
| `convert-onenote-to-markdown.ps1` | Windows (PowerShell) | Convert OneNote exports to Markdown |
| `convert-onenote-to-markdown.sh` | Unix / WSL | Same conversion for Unix environments |
| `transformFilesToMd.sh` | Unix / WSL | Batch-transform files to Markdown |
